Описание
Десктопное GUI-приложение на Python (tkinter) для ведения личной коллекции фильмов.
Позволяет добавлять фильмы, просматривать их в виде таблицы, фильтровать по жанру и году выпуска, а также сохранять и загружать данные в формате JSON.

Функционал
- Добавление фильмов с валидацией полей
- Табличный вывод коллекции
- Фильтрация по жанру (частичное совпадение) и году (точное совпадение)
- Сохранение/загрузка данных через `movies.json`
- Проверка ввода: год — целое число, рейтинг — дробное/целое от 0 до 10

1. Установите Python 3.7+
2. Склонируйте или скачайте проект
3. Запустите из терминала:
   ```bash
   python movie_library.py
